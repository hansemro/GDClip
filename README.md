# GDClip: Clip + Godot

Language: GDNative (C++)

Tested on Godot Engine v3.4.3 (Linux/X11)

`clip` is a cross-platform copy/paste clipboard library with image support. This
demo leverages this library to enable image-paste functionality into Godot.
